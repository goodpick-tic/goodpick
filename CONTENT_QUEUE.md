# CONTENT_QUEUE

GOODPICKの記事ネタを管理するストックリスト。毎晩21:00の働きかけでは、このリストから未着手のアイテムを1つ選んで提案する。
リストが空、または全アイテムが完了している場合は、村田さんに直接「今日は何について書きますか？」と尋ねる。

## ステータス凡例
- [ ] 未着手
- [~] 進行中（インタビュー中・ドラフト中）
- [x] 公開済み

## オーナーレビュー（所有物レビュー）

- [x] サロモン XA 3D PRO GTX — 公開済み（2026-05-24）
- [ ]

## ライフスタイルカタログ記事
- [x] 春山・GWキャンプに向けて今買っておきたいギア — 公開済み
- [ ]

## 運用メモ
- 新しいネタが出たら、このファイルに追記する
- 1つの記事が完了したら、ステータスを `[x]` に更新してコミットする
- オーナーレビューとライフスタイルカタログのどちらを優先するかは固定ルールにせず、村田さんのその時の状況（ネタの有無・気分）に応じて毎回判断する
